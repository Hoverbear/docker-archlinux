### Dependencies ###
You'll need:

* `gpg`
* `curl`
* `docker`
* 64-bit Linux 2.6.23 or higher.

This should work on almost any distro.

### Building ###
Pull the script, run it.

Change the `$USER` at the bottom if you'd like to have it under a different name.
